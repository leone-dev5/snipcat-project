# Observações do projeto em questão do layout.

![Exemplo:](/public/example/example.png)


- A interface dele é muito familiar de outros editores mas essa é a interface inicial.

- Algumas funções estão em desenvolvimento então elas estão bloqueadas ou não existem ainda.

- O snipcat editor é foi criado com intenção de ser totalmente gratuito ao usuário, diferente de outros web apps de edição esse é totalmente sem anúncio.

# Então falando do código e os recursos que até o momento estão desenvolvidos em alfa.

## Renderização:

# Funcionalidades:

- Devido a problemas técnicos com área de trabalho o editor só possui poucas funções como cortar, move, diminuir, aumentar imagens.

- Umas das funções é Linhas métricas.

![Exemplo:](/public/example/example_2.png)

- Quando essa função é ativar ela abre às linhas métricas

- Para acionar é só ir até adicionais e clicar em configurações depois ativar as linhas métricas.
